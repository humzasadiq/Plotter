# Vector Plotter

Graph Vector Plotting Application. 
  
![plot](https://github.com/humzasadiq/Plotter/blob/main/media/plot.gif?raw=true)  
  
**Keybinds**  
* Ctrl-+ = Add vector  
* Crtl-Enter = Plot vector  
* Crtl-minus = Exit  
* Crtl-* = Dot Product  
* Crtl-1/2/3 = Move through input boxes
